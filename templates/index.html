About This Tool

This web app predicts the probability of heart disease using clinical inputs (screening support only).

Logistic Regression
Fast baseline classifier that outputs probabilities.
Random Forest
Ensemble of many decision trees to improve robustness.
SVM
Finds a separating boundary; can work well in high dimensions.
Gradient Boosting
Sequential trees that correct earlier errors (boosting).
Medical Disclaimer: This is a screening tool only, not a medical diagnosis. Always consult a qualified healthcare professional.
